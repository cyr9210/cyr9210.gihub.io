### 배열
- 열의 생성과 선언에 관련하여 수강하였다.
- 향상된 for 문

```
for(데이터타입 변수명 : 배열이름){   
실행문 변수명 일력시 배열값 순서대로 호출(배열의 크기만큼)   
}
```
![Java](/images/java/java04-01.png)
- 향상된 for문 구현방법
- length메소드 사용방법
- 배열의 index값은 0 부터 시작됨
